## Code Standards

<div align="center">

<table>
<tr>

## Commit Message Format

```
type: brief description

Detailed explanation if needed.
```

<div align="center">

| Type | Purpose |
|---|---|
| `feat:` | New feature |
| `fix:` | Bug fix |
| `docs:` | Documentation update |
| `style:` | Formatting, no code change |
| `refactor:` | Code restructuring |
| `test:` | Adding or updating tests |
| `chore:` | Maintenance tasks |

</div>

## Getting Started

### Prerequisites

<div align="center">

| Tool | Version |
|---|---|
| Node.js | 18+ |
| npm or yarn | Latest |
| Git | Latest |

</div>

### Local Development

```bash
# Clone the repository
git clone https://github.com/QZENOfficial/QZEN.git
cd QZEN

# Install dependencies (website)
cd website
npm install
npm run dev
```
